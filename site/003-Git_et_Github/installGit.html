Installation de Git

Sommaire

Vérifier l'installation de Git

La commande ci-dessous, renvoi le numéro de la version installée.

git --version

Configurer Git

Configurer le nom de l'utilisateur

git config --global user.name "Prenom Nom"

Configurer l'e-mail de l'utilisateur

git config --global user.email email@domaine.extension
