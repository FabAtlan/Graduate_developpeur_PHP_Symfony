Qu'est-ce qu'un objet JavaScript ?

Pour manipuler des données complexes, il faut utiliser les objets (les informations d'un utilisateur : firstName, lastName, email, age, phoneNumber...).

Un objet contient des données sous forme de propriété / valeur.

Contrairement aux tableaux, un objet possède des propriétés (firstName...).

Déclarer un objet avec Object()

Un objet se déclare avec le constructeur Object() à l'aide d'une variable et lui affecter comme valeur un nouvel objet (new Object()).

Ses différentes propriétés seront accolées à cette variable par un point, puis le nom de la propriété désirée et lui affecter une valeur à droite du signe =.

let user = new Object()
user.firstName = 'John'
user.lastName = 'DOE'
user.age = 36
user.email = 'j.doe@email.com'
user.phoneNumbers = ['0660504030', '0123456789']
user.fullName = function() {
return this.firstName + ' ' + this.lastName
}

console.log(user)

Affichage console :
{ firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: [ '0660504030', '0123456789' ],
fullName: [Function]
}

this

Dans la fonction ajoutée à l'objet, le mot-clé this se réfère à l'objet courant au sein duquel le code est écrit (ici, il se réfère donc à : user).

Dans le cas d'un nouvel objet user2, this se référera cette fois à user2.

Déclarer un objet avec la notation littérale

La notation littérale peut être utiliser pour créer un objet en listant les propriétés entre accolades.

let user = {
firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: ['0660504030', '0123456789'],
fullName: function() {
return this.firstName + ' ' + this.lastName
}
}
console.log(user)

Affichage console :
{ firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: [ '0660504030', '0123456789' ],
fullName: [Function]
}

Créer une fonction pour construire nos objets

Il est possible de créer des objets avec une fonction, ce qui permet de les réutiliser pour faciliter leur utilisation et la modification de nos objets.

Par convention, le nom de la fonction commencera par une majuscule : User().
Cette fonction prendra en paramètres les différentes propriétés et valeurs.

Les valeurs seront passées à la fonction lors de son appel (ex : new User('John', 'DOE', ...)).

function User(
firstNameValue,
lastNameValue,
ageValue,
emailValue,
phoneNumbersValue
) {
this.firstName = firstNameValue
this.lastName = lastNameValue
this.age = ageValue
this.email = emailValue
this.phoneNumbers = phoneNumbersValue
this.fullName = function() {
return this.firstName + ' ' + this.lastName
}
}
let user = new User(
'John',
'DOE',
36,
'j.doe@email.com',
['0660504030', '0123456789']
)

console.log(user)

Affichage console :
User {
firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: [ '0660504030', '0123456789' ],
fullName: [Function] }

Les propriétés d'un objet JavaScript

Les propriétés

Une propriété Object() peut stocker tous les types (string, number, booleen, Array()).

Avec le point (.)

Pour accéder à une propriété, utilisez le nom de la variable suivi d'un point, puis du nom de la propriété.

Si la valeur de la propriété est un tableau, alors nous pouvons la manipuler comme tel, par exemple en affichant la première valeur (user.phoneNumbers[0]).

Si la valeur de la propriété est une fonction, alors il faut lui ajouter des parenthèses pour l'appeler user.fullName().

let user = {
firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: ['0660504030', '0123456789'],
fullName: function() {
return this.firstName + ' ' + this.lastName
}
}

// Display : John
console.log(user.firstName)

// Display the first phone number in the array : 0660504030
console.log(user.phoneNumbers[0])

// Display the return of the fullName() function : John DOE
console.log(user.fullName())

Affichage console :
John
0660504030
John DOE

Avec les crochets ([])

Il est également possible de manipuler les propriétés à la manière d'un tableau en accolant à la suite de notre variable le nom de la propriété entre crochets. Le nom de la propriété devra être mis entre apostrophes au sein des crochets (user['firstName']).

let user = {
firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: ['0660504030', '0123456789'],
fullName: function() {
return this.firstName + ' ' + this.lastName
}
}

// Display : John
console.log(user['firstName'])

// Display the first phone number in the array : 0660504030
console.log(user['phoneNumbers'][0])

// Display the return of the fullName() function : John DOE
console.log(user['fullName']())

Affichage console :
John
0660504030
John DOE

Modifier une valeur de propriété

Pour modifier une valeur, appelez l'une des méthodes et lui affecter une nouvelle valeur.

let user = {
firstName: 'John',
lastName: 'DOE'
}

user.firstName = 'Bob'

// Display : Bob
console.log(user.firstName)

Affichage console :
John
0660504030
John DOE

Ajouter une nouvelle propriété

Pour ajouter une nouvelle propriété, nous utilisons la variable, puis la propriété avec la méthode entre accolades et nous lui affectons la valeur désirée.

let user = {
firstName: 'John',
lastName: 'DOE'
}

user.userName = 'jdoe'

// Display : { firstName: 'John', lastName: 'DOE', userName: 'jdoe' }
console.log(user)

Affichage console :
{ firstName: 'John', lastName: 'DOE', userName: 'jdoe' }

Différences entre l'objet Object() et l'objet Array()

Similitudes et différences

L'objet Array() peut être assimilé à une liste. Pour récupérer un élément de la liste, on va itérer sur ses index tandis que pour un Object() on va itérer sur ses propriétés.

const carInObject = {
type: 'BMW',
doors: 3,
color: 'blue'
}

const carInArray = ['BMW', 3, 'blue'];

console.log(carInObject.color); // blue
console.log(carInArray[2]); // blue

Affichage console :
blue
blue

Un Object() peut stocker un Array(), et inversement. L'exemple ci-dessous renvoi undefined sur l'objet CarInObject.

const carInObject = {
type: 'BMW',
doors: 3,
color: 'blue'
}

const carInArray = ['BMW', 3, 'blue'];

console.log(carInObject.length);
console.log(carInArray.length);

Affichage console :
undefined
3

Pour connaître le nombre d'éléments présents dans un Object() il faut le transformer en Array().

Malgré leurs similitudes, les cas d'utilisation d'un Object et d'un Array sont très différent.

Un Array, au sens JavaScript est un objet mais un objet un peu particulier.

  • Un Array() sert à stocker une liste d'éléments.
  • Un Object() sert à stocker les caractéristiques d'un élément.

const car1 = {
type: 'BMW',
doors: 3,
color: 'blue'
}

const car2 = {
type: 'PEUGEOT',
doors: 5,
color: 'grey'
}

// Dans l'objet garage nous sotckons la liste des objets car
const garage1 = {
cars: [car1, car2]
}

const garage2 = {
cars: [car1, car2]
}

// Pour stocker la liste des garages nous utiliseront un Array()
const garages = [garage1, garage2]

Les méthodes de l'objet Object()

Transformer un Objet() en Array()

Il sera parfois nécessaire de transformer notre Objet en Array.
Nous allons citer quelques méthodes pour y parvenir :

  • Object.keys() : permet de récupérer un tableau contenant la liste des propriétés de l'objet donné.
  • Object.values() : permet de récupérer un tableau contenant la liste des valeurs des propriétés de l'objet.
  • Object.entries() : renvoie un tableau à plusieurs dimensions contenant les propriétés avec les valeurs associées.

let user = {
firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: ['0660504030', '0123456789'],
}

console.log(Object.keys(user));
//['firstName', 'lastName', 'age', 'email', 'phoneNumbers']
console.log(Object.values(user));
// ['John', 'DOE', 36, 'j.doe@email.com', ['0660504030', '0123456789']]
console.log(Object.entries(user));
[[ 'firstName', 'John' ],
[ 'lastName', 'DOE' ],
[ 'age', 36 ],
[ 'email', 'j.doe@email.com' ],
[ 'phoneNumbers', [ '0660504030', '0123456789' ] ],
[ 'fullName', [Function] ]]

HasOwnProperty()

Afin d'éviter de transformer un Object en Array, nous pouvons vérifier qu'une propriété existe avec Object.hasOwnProperty('property').

let user = {
firstName: 'John',
lastName: 'DOE',
age: 36,
email: 'j.doe@email.com',
phoneNumbers: ['0660504030', '0123456789'],
}

let propertyLastNameExist = user.hasOwnProperty('lastName')
if (propertyLastNameExist) {
console.log(user['lastName'])
}
//DOE
