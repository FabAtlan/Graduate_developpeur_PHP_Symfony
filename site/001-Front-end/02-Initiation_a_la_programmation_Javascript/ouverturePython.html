Introduction au langage Python

La factorielle en mathématiques

Dans le Web en particulier, on peut trouver des langages tels que JavaScript côté client et d'autres tels que PHP, Java ou .NET pour la partie serveur. Python permet aussi de réaliser du développement web côté serveur.

Rappel :

La factorielle d'un entier naturel n est le produit des nombres entiers strictement positifs inférieurs ou égaux à n.
Ainsi, pour calculer la factorielle de 5 (aussi noté 5!), on obtient 5 * 4 * 3 * 2 * 1, soit 120.

Syntaxe Python

#!/usr/bin/env python3
n = int(input('Entrez un nombre, et sa factorielle sera affichée : '))

if n < 0:
raise ValueError('Vous devez saisir un nombre positif')

fact = 1
i = 2
while i <= n:
fact *= i
i += 1

print(fact)

Equivalent JavaScript :

const n = parseInt(prompt('Entrez un nombre, et sa factorielle sera affichée: '))
if (n < 0) {
throw 'Vous devez saisir un nombre positif'
}

let fact = 1
let i = 2

while (i <= n) {
fact *= i
i += 1
}
document.write(fact)

Les variables et les opérateurs

Affecter une variable

En Python, il n'existe pas vraiment de mot-clé qui permet de déclarer une variable, il suffit de simplement l'utiliser.

x = 10

Les différents types de données

  • texte : str
  • numériques : int, float, complex
  • séquences : list, tuple, range
  • mapping : dict
  • set : set, frozenset
  • booléen : bool
  • binaires : bytes, bytearray, memoryview

Une variable de type texte

Il faut délimiter une chaîne de caractères directement avec des " ou en utilisant la syntaxe avec str() :

message = "Hello world"
print(type(message))
# ou
message = str("Hello world")
print(type(message))

La notation des booléens

La définition d'une valeur booléenne prend une majuscule : True ou False. Écrites true ou false, ces chaînes seront considérées comme des variables et non des valeurs.

true = True

Les commentaires

En Python, les lignes de commentaires commencent par un #. La syntaxe pour un commentaire multi-lignes se fait avec 3 """

#This is a
#multiline comment

"""
This is a
multiline comment
"""

Opérations de concaténation

En Python, il existe plusieurs formes de concaténation :

  • avec le caractère +
  • avec une syntaxe avec tokens.

Pour concaténer deux chaînes en JavaScript, on utilisera la syntaxe suivante :

message = "Hello " + "world"

Pour l'expression string literal :

hello = "Hello"
world = "world"
message = "%s %s" % (hello, world)

Opérations arithmétiques

  • +, -, *, / ou %
  • ** calcule l'exponentiel d'un nombre
  • // calcule le quotient entier arrondi à l'inférieur d'une division, par exemple 19 // 5 donnera 3.

Les conditions et les boucles

Opérateurs de comparaison

  • L'égalité : ==
  • L'inégalité : !=
  • La supériorité : >
  • L'infériorité : <
  • Et leur déclinaison avec la nuance d'égalité : >= et <=

Opérateurs logiques

  • and qui retourne vrai, si les deux conditions sont vraies
  • or qui retourne vrai, si l'une des deux conditions est vraie
  • not qui retourne vrai, si la condition est fausse

    • Opérateurs d'identité

    • in qui retourne vrai, si la valeur est trouvé dans l'objet
    • not in qui retourne vrai, si la valeur est absente de l'objet

    Construire une expression booléenne

    expression = x > 5 and y <= 10

    Structure conditionnelle

    if b > a:
    print("b est plus grand que a")
    else:
    print("a est plus grand que b")

    Le mot-clé elseif devient elif en Python.

    if b > a:
    print("b est plus grand que a")
    elif a == b and b > c:
    print("a et b sont égaux et b est plus grand que c")

    Imbriquer des structures conditionnelles

    if x > 0:
    print("x est positif ")
    if x = 0:
    print("et égal à 0")
    else:
    print("strictement")

    Boucles

    Il existe deux types de boucles en Python : for et while.

    Boucle for

    users = ["Nicolas", "Laurent", "Laure"]
    for user in users:
    print(user)

    Boucle while

    users = ["Nicolas", "Laurent", "Laure"]
    i = 0

    while i < len(users):
    print(users[i])
    i += 1

    Les fonctions

    Appeler une fonction native

    On appelle fonctions natives les fonctions incluses dans un langage. Il en existe beaucoup en Python.

    • Les fonctions de conversion de type appelé cast : int(), str(), bool(), etc.
    • input() qui permet de récupérer une saisie utilisateur
    • print() qui permet d'afficher un texte à l'écran
    • len() qui permet de récupérer la longueur d'un objet (chaîne de caractères, tableau, etc.)
    • min() et max() qui permettent de récupérer respectivement la valeur minimale et maximale d'un tableau

    Récupérer la longueur d'un objet

    string = "Hello world"
    length = len(string)
    print("La chaîne \"%s\" fait %s caratères de long" % (string, length))

    Afficha console :
    La chaîne "Hello world" fait 11 caractères de long

    Les modules

    On peut accéder à d'autres fonctions présentes dans des modules importés avec le mot-clé import. La ligne d'import d'un module doit être positionnée tout en haut du script.

    import random
    print(random.randint(0, 100))

    Déclarer une fonction

    Afin de déclarer une fonction en Python, on va utiliser le mot-clé def. La syntaxe est identique aux structures conditionnelles et aux boucles.

    def say_hello(name, age = 0):
    print("Bonjour " + name)
    if age > 0:
    print("Tu as %s ans" % (age))

    say_hello("Nicolas")

    Retourner une valeur

    Si on veut pouvoir retourner une valeur, on utilise le mot-clé return, comme dans les autres langages.
