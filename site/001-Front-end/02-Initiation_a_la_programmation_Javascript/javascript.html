Les objets

Exemple :

window est un objet fourni par le navigateur.

console est un objet.

Qu'est-ce qu'un objet ?

Voiture :

Un objet a des propriétés/attributs et des méthodes.

  • Caractéristiques (couleur, puissance...) : propriétés/attributs
  • Actions (démarrer, accélérer,...) : méthodes.

Syntaxe pour accéder à une méthode :
objet.methode()

Syntaxe pour accéder à une propriété :
objet.propriete

Créer un objet

var product1 = {nom: 'Table', ht: 100}
var product2 = {nom: 'Couteau', ht: 5}
console.log(product1.nom + ' prix ht = ' + product1.ht)
console.log(product2.nom + ' prix ht = ' + product2.ht)

Ajouter une propriétés

product1.color = 'red'

Les tableaux ARRAY

Déclarer un tableau

var myTab = new Array()

var myTab = []

Les tableaux sont indexés de 0 à nombre d'entrées - 1.

Récupérer le nombre total d'éléments dans un tableau

var nElements = myTab.length

Ajouter un objet à la fin d'un tableau

myTab.push(element)

Supprimer le dernier élémet d'un tableau

myTab.pop()

Supprimer un élément d'un tableau

delete myTab[1] // supression de la valeur qui sera a undefined

Parcourir un tableau

for (let i in myTab) {...}
