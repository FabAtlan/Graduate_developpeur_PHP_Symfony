Comment utiliser jQuery ?

La fonction principale

Pour utiliser cette librairie la fonction principale doit obligatoirement être incluse

jQuery(document).ready(() => {
// Code jQuery
})

document représente le DOM.
ready est l'événement signifiant que le code s'exécutera une fois la page entièrement chargée.

Ecriture simplifiée de la fonction principale avec l'alia $

$(function() { // ready est remplacé par l'alias $
// Code jQuery
});

Si l'alias $ est utilisé par d'autres librairies, préférez les écriture ci-dessous pour bien identifier la librairie jQuery.

jQuery(function($){
alert('jQuery est inclus !')
});
// Ou
$(function($){
alert('jQuery est inclus !')
});

Syntaxe générale

$('selecteur'). methode()
  • $ doit être systématiquement utilisé.
  • 'selecteur'sélectionnez l'élément du DOM
  • methode()indiquez la méthode afin de réaliser le traitement

Les sélecteurs jQuery

La syntaxe de base est la suivante :

$('selecteur').methode()

Les sélecteurs jQuery s'utilisent de la même façon qu'en CSS. Une sélection par identifiant commence par # et pour une classe avec un ..
Exemple :

$('div') // balise
$('#bases') // identifiant
$('.bases') // classe

Enchaîner les sélecteurs

$('div p') // Séléction de la balise div puis de la balise p
$('#content p') // Séléction de l'identifiant content puis de la balise p

Sélection avec les pseudo-classes

$('#content:first-child') // Sélectionnera le 1er enfant de l'élément #content.

Liste des pseudo-classes les plus utilisées :

  • :first-child
  • :last-child
  • :last-of-type
  • :hover
  • :before
  • :after
  • nth-child(x)

Sélection multiple

Listez les sélecteurs en les séparant par une virgule.

$('h1, h2, h3, .titre') // Sélectionnera les balises h1, h2 et h3 ainsi que les éléments ayant la classe .titre

Le sélecteur this

Permet d'agir sur l'élément courant mais il n'est utilisable qu'à l'intérieur d'une fonction anonyme.

$('#content p').text('Bonjour tout le monde !')

$('#content p').html(function() {
$(this).text('Bonjour tout le monde !')
})

D'un point de vue performance, préférez pour le ciblage d'un élément du DOM, la notion de parent/enfant.

La programmation événementielle

  • html() permet de récupérer le code HTML d'un élément. Si par contre un argument lui est affecté, cette méthode ajoutera du contenu HTML.
    $('p').html() // Retourne le contenu HTML
    $('p').html('<p>Hello !</p>') // Ajoute un nouveau contenu HTML
  • css('key', 'value') permet d'ajouter/modifier la valeur d'un attribut CSS.
    $('h3').css('Propriete', 'valeur')
  • each() permet de parcourir une liste d'éléments. Elle prend en paramètre une fonction anonyme :
    <div id="content">
    <ul id="list">
    <li>PHP</li>
    <li>HTML</li>
    <li>CSS</li>
    <li>jQuery</li>
    </ul>
    span.code</div>

    $(document).ready(() => {
    $('#list li').each(function() {
    console.log($(this).text())
    })
    })
  • data() : permet de créer des objets utilisables seulement par jQuery.
  • text('exemple') : insert du texte dans l'élément sélectionné.
  • click() : écouter l'événement click de la souris.

Ajouter un élément

append() permet d'ajouter un élément en tant que dernier enfant.

prepend() permet d'ajouter un élément en tant que premier enfant.

<div id="content">
<ul id="list">
<li>Au milieu</li>
</ul>
</div>

$(function() {
let list = $('ul')
list.prepend('<li>Premier</li>')
list.append('<li>Dernier</li>')
});

// Résultat navigateur
Premier
Au milieu
Dernier
