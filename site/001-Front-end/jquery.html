Installer jQuery en minifié

Comment utiliser jQuery ?

La fonction principale

Pour utiliser cette librairie la fonction principale doit obligatoirement être incluse

jQuery(document).ready(() => {
// Code jQuery
})

document représente le DOM.
ready est l'événement signifiant que le code s'exécutera une fois la page entièrement chargée.

Ecriture simplifiée de la fonction principale avec l'alia $

$(function() { // ready est remplacé par l'alias $
// Code jQuery
});

Si l'alias $ est utilisé par d'autres librairies, préférez les écriture ci-dessous pour bien identifier la librairie jQuery.

jQuery(function($){
alert('jQuery est inclus !')
});
// Ou
$(function($){
alert('jQuery est inclus !')
});

Syntaxe générale

$('selecteur'). methode()
  • $ doit être systématiquement utilisé.
  • 'selecteur'sélectionnez l'élément du DOM
  • methode()indiquez la méthode afin de réaliser le traitement

Les sélecteurs jQuery

Les sélecteurs jQuery sont les mêmes qu'en CSS et s'utilisent de la même façon.

<div id="content">Présentation de jQuery</div>
$('#content')

Autres méthodes de sélection.

$('div') // Sélection de la div par son nom de balise.
$('#content') // Sélection de la div par son attribut id #content
$('.content-div') // Sélection des div par la classe .content-div

Appeler une méthode

A la suite d'une méthode

  • html('<span>exemple</span>') : insert du code HTML dans l'élément sélectionné.
  • text('exemple') : insert du texte dans l'élément sélectionné.
  • css('key', 'value') : ajouter/modifier la valeur d'un attribut CSS.
  • click() : écouter l'événement click de la souris.

Avec une fontion anonyme

Possibilité d'y associer un script avec écouteurs d'événements.

Avec enchaînement des méthodes

Permet d'associer plusieurs comportements à un élément en une seule fois.

$('#content p').text('Bonjour tout le monde !').css('color', 'blue')
