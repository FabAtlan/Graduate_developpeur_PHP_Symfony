Manipuler les champs de type texte

Le formulaire HTML étant en place, JavaScript va nous permettre d'interagir avec lui.

Accès et modification de la valeur d'une zone de texte

Pour accéder à la valeur d'un champ texte (text, textarea), utiliser la propriété value

<form>
<input type="text" id="short-txt" name="short-txt" value="un text court">
<textarea id="long-txt" name="long-txt">Un texte long</textarea>
</form>
//---------------
var shorttxt = document.getElementById('short-txt')
var longtxt = document.getElementById('long-txt')
// Accès à la valeur par la propriété value
console.log(shorttxt.value, longtxt.value)

Modification

var longtxt = document.getElementById('long-txt')
longtxt.value = 'Un texte modifié !'
console.log(shorttxt.value, longtxt.value)

Focus et blur sur une zone de texte

Nous pourrons lier deux événements spécifiques : focus et blur. focus est déclenché quand on clique dans une zone de texte pour y écrire.
blur est déclenché quand nous quittont un champ.
