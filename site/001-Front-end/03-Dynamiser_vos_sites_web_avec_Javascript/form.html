Manipuler les champs de type texte

Accéder à un champ texte

Pour accéder à la valeur d'un champ texte (text, textarea), utiliser la propriété value

<form>
<input type="text" id="short-txt" name="short-txt" value="un text court">
<textarea id="long-txt" name="long-txt">Un texte long</textarea>
</form>
//---------------
var shorttxt = document.getElementById('short-txt')
var longtxt = document.getElementById('long-txt')
// Accès à la valeur par la propriété value
console.log(shorttxt.value, longtxt.value)

Modification

var longtxt = document.getElementById('long-txt')
longtxt.value = 'Un texte modifié !'
console.log(shorttxt.value, longtxt.value)

Focus et blur sur une zone de texte

Nous pourrons lier deux événements spécifiques : focus et blur. focus est déclenché quand on clique dans une zone de texte pour y écrire.
blur est déclenché quand nous quittont un champ.

Haut

Document.createTextNode()

Créé un nouveau noeud de texte.

var text = document.createTextNode(data);

text est un nœud de texte.

data est une chaîne contenant les données à placer dans le nœud de texte.

Manipuler les checkbox, radio et select

Les éléments de formulaire donnant lieu à un choix de l'utilisateur ont en commun l'événement change.

Checkbox

La propriété checked de l'objet Event est utilisé pour détecter l'état du bouton.

//HTML
<label for="check">Cocher moi</label>
<input type="checkbox" id="check" name="check">

//SCRIPT
var checkbox = document.getElementById('check')
checkbox.addEventListener('change', () => {
alert(event.target.checked)
// True si coché, false sinon
})

Boutons radio

Avec les boutons radio seul un choix est possible. Il faut donc placer un addEventListener sur chacun des choix à l'aide d'une boucle.

//HTML
<input type="radio" name="gender" value="Femme">Femme
<input type="radio" name="gender" value="Homme">Homme

//SCRIPT
var radio = document.getElementsByName('gender')

for (var i = 0; i < radio.length; i++) {
radio[i].addEventListener('change', (e) => {
alert(e.target.value);
});
}

Listes déroulantes

De la même manière, l'événement change est employé.

//HTML
<label for="language">Langage :</label>
<select name="language" id="language">
<option value="javascript" selected>JavaScript</option>
<option value="php">PHP</option>
<option value="python">Python</option>;
</select>

//SCRIPT
var language = document.getElementById('language')

language.addEventListener('change', (event) => {
alert(event.target.value)
})
