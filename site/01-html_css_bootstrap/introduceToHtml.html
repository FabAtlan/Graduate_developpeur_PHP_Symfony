Introduction au HTML

Internet

Internet est un réseau de communication mondial entre ordinateurs, créé dans les années 60 pour l'armée américaine (ARPAnet).
Lorsque l'on se connecte sur un site web, on se connecte sur un autre ordinateur (appellé serveur ) relié au réseau Internet. Cet ordinateur a été programmé pour renvoyer le contenu du site à chaque ordinateur qui lui demande. Sur ce réseau, tous les ordinateurs peuvent communiquer entre eux en s'envoyant des messages. Pour se comprendre, il a fallu concevoir une même manière d'envoyer ces messages : c'est le protocole TCP/IP .

Il est la fusion de deux protocoles :

TCP (Transmission Control Protocol)
désigne la manière dont les messages doivent être écrits et leur formalisme.
IP (Internet Protocol)
désigne la manière d'attribuer un identifiant unique à un ordinateur pour pouvoir le retrouver.

Qu'est-ce qu'une adresse IP ?

C'est une adresse définie pour chaque ordinateur sur le réseau par le protocole. Les serveurs, étant aussi des ordinateurs, en ont également. Elle prend la forme de 4 nombres compris entre 0 et 255 séparés par des points. Par exemple : 21.245.3.98.

Qu'est-ce qu'un nom de domaine ?

Comme les adresses IP sont difficiles à mémoriser et peuvent être amenées à changer, il a été décidé d'utiliser un système plus simple : les noms de domaine.
Les noms de domaine sont composés :

  • d'un domaine de premier niveau (TLD, pour Top-Level Domain) ou extension. Les TLD se placent à la fin du nom de domaine et indiquent leur provenance (fr, be...) ou le type de site Internet (com, org...).
  • d'un domaine et, parfois, d'un ou plusieurs sous-domaines, tous séparés par des points. Les domaines sont le nom du site, comme "google" ou "wikipedia".
  • de sous-domaines pouvant définir des sous-parties d'un même domaine (mail, drive...).

Le DNS

Il existe un système qui transforme le nom de domaine en adresse IP. Ce système est appelé le DNS , pour Domain Name System. Les serveurs DNS sur le réseau possèdent une liste de noms de domaines et des adresses IP associées. Il suffit de leur donner un nom de domaine pour qu'ils retournent l'adresse IP correspondante.

Découverte du Web

Le Web actuel, tel que nous le connaissons, a été inventé par Tim Berners-Lee en 1989, chercheur au CERN, afin de rendre les échanges entre les membres de la communauté scientifique instantanés. Le World Wide Web (WWW), le Web et la Toile sont en fait la même chose : il s'agit d'une application mondiale qui fonctionne grâce à Internet et qui met à notre disposition des milliards de ressources.
Il faut donc bien distinguer le Web des sites web. Le Web est comme une bibliothèque géante que l'on peut parcourir, un site est un livre, et une page web est une page du livre.

Internet vs Web

On entend souvent parler de site Internet, mais c'est un abus de langage. Il s'agit de deux choses complètement différentes. Le Web est l'application permettant de consulter des données, représentées sous forme de sites web, qui sont disponibles sur Internet.

Les sites web

Un site web est un ensemble de pages web, hébergées sur un serveur. Les trois plus répandues sont :

  • les sites de vente en ligne dit de e-commerce,
  • les sites vitrine donnant aux entreprises plus de visibilité sur leur activité ou leurs produits,
  • l'application web permettant de réaliser des traitements complexes.

Les composants d'un site

Un site est un ensemble de données :

  • du texte, avec diverses fonctionnalités (liens vers d'autres pages, titres...) utilisant le langage HTML,
  • des feuilles de style pour l'apparence, utilisant le langage CSS,
  • des fichiers multimédias comme des vidéos ou du son, aussi importés en HTML,
  • du contenu interactif grâce à des scripts en JavaScript, permettant aux pages de devenir plus vivants et plus complexes.

Back-end vs Front-end

Le back-end, est la partie du code exécuté par le serveur pour générer nos pages web.
Le front-end, est la partie visible d'un site web, soit le code exécuté par le navigateur.
Le HTML, le CSS et le JavaScript sont ainsi les trois grandes composantes de ce que doit connaître un développeur front-end. Pour le back-end, le langages le plus utilisé est le PHP.

Les différents types de site

Il existe deux grands types de site :

  • site statique : c'est un site qui ne change jamais.
  • site dynamique : c'est un site qui peut se mettre régulièrement à jour.

Fonctionnement des sites

Afin de mettre un site à jour, on utilise un serveur web. C'est l'ordinateur qui va stocker les données relatives au bon fonctionnement des sites qu'il héberge, dont le code. Il assure également la disponibilité des ressources (images, les vidéos...) ! Lorsque l'on effectue une recherche, notre navigateur (le client) envoie alors une demande à l'ordinateur qui stocke le code du site (le serveur). On effectue alors une requête. Le serveur va alors nous renvoyer la ressource que nous lui avons demandée, puis notre navigateur se charge de l'afficher. On dit alors qu'il nous envoie une réponse. On appelle ce type de communication une communication client/serveur.
Il est possible, dans le cas d'un site dynamique, qu'il fasse appel à une base de données afin de récupérer des données supplémentaires : par exemple en cas d'une connexion, il affiche les données qui nous sont propres.
